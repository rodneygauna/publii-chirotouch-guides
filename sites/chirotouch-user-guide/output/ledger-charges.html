Patient Ledger - Charges 

The Charges card in the Patient Record displays all patient-related charges. Users can manage all patient charges which include adding, editing, voiding, and allocating charges to payments. 

Use the filters to narrow your search. The Date of Service automatically defaults to the last 30 days. 

Add New Charge

Charges first need to be created in Billing Settings - Charges. Click here to learn more about Billing Settings - Charges.

1.  Select Add New Charge (in blue).

2.  From the Add Charge slideout, select one of the following charges:

  • Purchase - products sold to the patient, to include DME, that will not be billed to insurance
  • Fee - an additional charge, such as a late fee, that can be appended to an invoice
  • Encounter - additional procedures that can be added to an encounter

Add Purchase

Products sold to the patient that will not be billed to insurance.

1.  From the Add Charge slideout, select Purchases (in blue).

2.  From the Add Charge - Purchase slideout, select Add Charge (in blue).

3.  Enter code or description to filter the list. Once the desired charge is found, select Add Charge, followed by DONE (in blue).

4.  Update the charge details as necessary:

  • Units - number of products
  • Adjustment - select an adjustment code if applicable
  • Amount - if applicable 
  • Tax - select a tax rate if applicable 
    • None
    • City
    • State

Note: Adjustment Codes and Tax rates first have to be created in Billing Settings. Click here to learn more about Adjustment Codes and Tax Rates.

5.  Choose invoice details.

  • Generate New Invoice - generate a new invoice with this charge
  • Add to Existing Invoice - add this charge to an existing invoice
  • Guarantor - guarantor responsible for this charge
    • Self - indicates the individual patient
  • Invoice - the invoice used to add this charge
  • Attribute To - select a provider who this purchase is attributed to

6.  Select PAY, POST, or CANCEL.

  • PAY - posts the charge to the patient's ledger while allowing you to take payment simultaneously 
  • POST - posts the charge to the patient's ledger without payment 

Add Fee

An additional charge, such as a late fee, can be added to an invoice.

1.  From the Add Charge slideout, select Fee (in blue).

2.  From the Add Charge - Fee slideout, select Add Charge (in blue).

3.  Enter code or description to filter the list.  Once the desired charge is found, select Add Charge, followed by DONE (in blue).

3.  Update the charge amount. Select PAY, POST, or CANCEL. 

  • PAY - posts the charge to the patient's ledger while allowing you to take payment simultaneously 
  • POST - posts the charge to the patient's ledger without payment 

Add Encounter Charge

Additional procedures that can be added to an encounter. Encounter charges require an encounter to be created first in Charting. 

1.  From the Add Charge slideout, select Encounter Charge (in blue).

2  From the Add Charge - Encounter slideout, select the encounter you wish to add a charge. 

3.  Existing encounter charges will display. Click Add Charge (in blue) to add to this encounter.

3.  Enter code or description to filter the list. Once the desired charge is found, select Add Charge, followed by DONE (in blue).

Note: You can add single, multiple, or all charges. 

4. Once you have added all the charges, click POST (in blue) to add them to the patient encounter selected.

5. Both the Charges and Balance cards update accordingly. 

View Charges

The Charges page displays all charge-related information along with actions that can be taken regarding those charges. You can view charges either by using the action menu or by expanding the charge details.

1a. action menu: Select the checkbox(es) next to the charge(s) you would like to manage, then click the action menu (ellipsis). Select View Charges. 

OR

1b.  Select the Charge you wish to view. Once expanded, select the right arrow.

2. From the ellipsis, you can View Charges or use the arrow on the far right to open the charge details.

3.  From Charges page, view available actions indicated (in blue).  Actions in gray are unavailable for the charge selected.

  • Edit Icon - change the payer responsibility (self vs. insurance), add a secondary payer, update the guarantor, add a case, or authorization
  • Add New Charge - add new charges to the existing charge invoice 
  • Edit Charges - update all charge billing information
  • Allocate Payment - allocate the charges to available payments 
  • Create Invoice - create an invoice for charges if one is already created
  • Remove From Invoice - remove the charges from the existing invoice 
  • Void Charges - voids the charges
  • Unpost - unposts the charge

Edit Charge

Editing a charge allows you to update all charge-related billing information. The example below is a self pay encounter office visit charge. You can either get there through the action menu (ellipsis) or by clicking on the charge itself.

1a. Select the Charge checkbox you wish to edit.  Click the action menu (ellipsis). 

OR

1b.  Select the charge you wish to edit. Click the right arrow to expand and check the box of the charge to edit.

2.  Select Edit Charges.

3.  From the Charges page, select the Charge you wish to edit, followed by Edit Charges (in blue).

4.  From Update Charges, the editable charge details will display on a scrollable horizontal charge table.

All the information related to a charge can be seen in Update Charges. You must scroll along the charge table to see and update the charge details. Below you will see the editable details broken into two sections. 

a. Charge Line Details: The first section includes information about the charge. Click each screenshot to enlarge.

    • Charge # - each has a unique number for identification
    • Encounter - each encounter has a unique number
    • Provider - the service provider
    • Dx- select applicable diagnosis chosen in the encounter
    • DOS - Date of Service
    • POS - allows you to adjust the Place of Service code 
    • Proc - procedure name
    • Modifiers - adjust or add modifiers related to this charge
    • 24 Shaded - pushes the information to FL 24 Shaded in the claim form
    • NDC - National Drug Code
    • Attribute To - select the provider you wish to attribute this charge to 
    • Units - adjust the units for this charge
    • Unit Charge - displays the unit charge amount used to calculate the billed amount
    • Fee Sch Chrg - Optional amount which can be used to override the unit charge, calculating the new billed amount
      • Billed - will display the billed amount 
      • Tax - Tax amount 
      • Allowed - Displays the allowed amount for the charge
    • EPSDT - Early & Periodic Screening Diagnostic Treatment
    • Fee Schedule - associated fee schedule for patient charges
    • Tax Rate - displays the tax rate for taxable items

b. More Charge Line Details: If you continue to scroll, you would see the payer and patient responsibility details, adjustments, and balance. Click the screenshot to enlarge and see the rest of the charge details.

  • Prim. Payer - the Primary Payer selected for this charge
  • Prim. Owed - the expected allowed amount reimbursement
  • Prim. Paid  - how much has been paid by the Primary Payer
  • Pymt ID(s) - the Payment ID associated with this charge
  • COB Checkbox - (Coordination of Benefits) - indicates a secondary payer responsibility 
  • Pt. Resp. Dropdown  - code used to denote the type of patient responsibility
  • Pt. Owed - indicates how much the Patient owes
  • Pymt ID(s) - the Payment ID associated with this charge
  • Adjustment Dropdown - code used to denote the adjustment type 
  • Amount - indicates how much is being adjusted 
  • Add Row - adds another responsibility row
  • Balance - displays the balance of the charge

5.  Once edits are made, select SAVE (in blue).

Unpost Charge

Once a charge has been posted, you will need to remove the charge from the invoice in order to unpost the charge.  Verify that the charge that you'd like to delete is not associated with a claim or draft.

Note: If the Encounter has been signed, you will need to reopen the encounter to remove the charge.

1.  Use the right arrow to open the invoice and show the associated charges.

2. Use the checkboxes to select the charge(s) that you would like to remove. Click the action menu (blue ellipsis).

3.  Select Remove From Invoice. 

4.  Select YES to continue.

5. Now you will see the charge(s) are no longer invoiced. 

6.  Use the action menu (ellipsis) and select View Charges.

7. Now check the box for the charge(s) that you would like to Unpost. 

NOTE: Unpost is not available if the chart note is signed.  You will need to return to the Encounter and reopen the note.

8.  When you Unpost the charge, you will receive a prompt acknowledging the charge(s) will be removed.

9.  You will see the green banner that verifies that the charge(s) have been unposted.  The charge(s) will still appear in the encounter and may be completely removed from there.

Invoices

Create Invoice

Below you will learn how to create an invoice for a charge. Once an invoice is created, you can add other or new charges to an existing invoice. You can either get there through the action menu (ellipsis) or by clicking on the charge itself.

1a.  Select the charge checkbox you wish to create an invoice. Select the action menu (ellipsis).

OR

1b.  Select the charge you wish to create an invoice. Use the right arrow to expand the charge.

2. Select Create Invoice.

3.  An invoice will be created for that charge.

Add Charge To Existing Invoice

You can add a charge to an existing invoice. 

1.  Select the checkbox you wish to Add To Invoice. Click the action menu (ellipsis).

2. Select Add To Invoice.

3. From Add to Invoice dialog, select the desired invoice from the dropdown, followed by ADD (in blue).

Note: Only invoices that match the charges' billing data of Guarantor, Case, Payers, and Plans may be selected. If you are unable to select an invoice, review its billing data to ensure your selected charges match these values. Edit the invoice or charge accordingly or select an alternate invoice.

4. The charge will be added to the selected invoice.

Remove From Invoice

Charges first have to be assigned to an invoice before being removed.  Verify that a charge is not associated with a claim or draft before you remove it.

1. To select the charge(s) you wish to remove, click the right arrow to expand the entire invoice.

2. Put a checkmark on the charge(s) you want to remove from the invoice.

3. Use the ellipsis to open the dropdown menu and select Remove From Invoice.

4. From Remove From Invoice dialog, select YES (in blue).

5. The charge has been removed from the current invoice and will show as it's own charge line with a status of Not Invoiced. 

Allocate Charge

To allocate a charge successfully, an unallocated payment matching the charge responsibility type needs to be added first. You can either get there through the action menu (ellipsis) or by clicking on the charge itself. Click here to learn more about Payments.

1a.  Select the charge checkbox you wish to Allocate, followed by the action menu (ellipsis).

OR

1b.  Select the charge you wish to Allocate.  Select the right arrow to expand the line.

2. Select Allocate Payment. 

3. From the Allocate Payments page, select the charge(s) you wish to allocate. 

4. From the Allocation slideout, the available payments will display at the top, selected by default. If there are multiple unallocated payments, select the desired payment.

Note: Payments that match the charge criteria will only be available to allocate. Charge criteria include responsibility type (patient or payer) and an available amount to allocate.

5.  Scrolling to the right on the bottom of charge table, find the responsibility section that corresponds to the payment. Update the Allocated column with the desired amount you wish to allocate.

Note: Charges can be partially or fully allocated.

6. Select ALLOCATE (in blue).

7. The Allocate Payments details will update.

Update Charge Responsibility

You can change the charge responsibility type from Patient or Self Pay, to Payer or Care Package. Users also can update the Guarantor and Case. You can either get there through the action menu (ellipsis) or by clicking on the charge itself.

Note: A Guarantor and Case must first be added to the patient's record. Click here to learn more about Guarantors and Cases.

1a.   Select the charge checkbox you wish to update, followed by the action menu (ellipsis).

OR

1b.  Select the charge line you wish to update.  To expand, select the right arrow. 

2. Click the ellipsis and choose View Charges.

3.  From the Charges page, select the Edit Icon (in blue).

4.  Edit the desired information which includes:

  • Bill Type - Self Pay (default), Care Package, Insurance
  • Payer - the patient's insurance payer
  • Plan - the plan associated with the selected insurance payer
  • Secondary - other insurance payer (secondary, tertiary, other) if applicable
  • Guarantor -guarantor
  • Case - the related patient case
  • Authorization - authorization related to patient case

5.  On a primary claim, the insurance payer listed in the Secondary column will populate boxes 9a and 9d on a claim form.  And there will be a Yes in box 11d.

6.  On a secondary claim, the insurance payer listed in the Plan column will populate boxes 9a and 9d on a claim form.  And there will be a Yes in box 11d.

Closed Tab

Zero balances charges are visible under the Closed tab in Charges.
