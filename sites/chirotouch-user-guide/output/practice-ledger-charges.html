Practice Ledger - Charges

The Charges card in the Practice Ledger displays all patient-related charges for the entire practice. From this screen, you can view, edit, and void charges and allocate payments to charges.

The Charge details can be filtered to quickly find specific charges or groups of charges. The Date of Service automatically defaults to the last 30 days.

View Charge Details

The Charges card in the Practice Ledger displays all patient-related charges for the entire practice. From this screen, you can view, edit, and void charges and allocate payments to charges.

The Action Menu allows you to make bulk selections and perform charge-related actions like create invoices, and view, edit, void, and allocate charges. 

1a.  Action Menu: Select the checkbox(es) next to the charge(s) you would like to manage, then click the action menu (ellipsis). Select View Charges.

 
OR

1b.  Expand the Charge Details: Scroll to the right and select the charge you would like to manage. Once the charge line expands, click the right arrow.

2.  From the Charges page, manage the charges using available actions:

  • Edit Icon - change the Payer responsibility, add a secondary Payer, update the Guarantor, and add a Case
  • Add New Charge - add a new charge to the existing charge invoice 
  • Edit Charges - update all charge billing information
  • Allocate Payment - allocate an available payment to the charges
  • Create Invoice - create an invoice if these charges are not already associated with one
  • Remove From Invoice - remove the selected charges from the existing invoice 
  • Void Charges - voids the selected charges
  • Unpost - unposts the selected charges (Note: Unposted charges are still associated with the encounter. To fully delete charges, return to the encounter to delete unposted charges.)

Edit Charges

Editing a charge allows you to update all billing information. You can edit charges either by using the action menu or by expanding the charge details.

1a. Action Menu: Select the checkbox(es) next to the charge(s) you would like to manage, then click the action menu (ellipsis). Select Edit Charges.

 
OR

1b. Expand the Charge Details: Scroll to the right and select the charge you would like to manage. Once the charge line expands, click the right arrow.

2.  From the Charges page, select the charge you wish to edit. Then click Edit Charges (in blue).

3. When the row expands to include all of the charge details, scroll right to see all of the editable options.

4.  Update the charge details. Editable options include:

  • Billed Status -  current status of claim (Patient, Primary Insurance, Secondary Insurance)
  • Hold Status - manually updated to show generation of Patient Statement, Superbill, Claim
  • Provider - view the provider associated with the encounter
  • Dx- select applicable diagnosis (Note: If no diagnosis was added to this encounter, this will not be an editable option.)
  • DOS - view the Date of Service
  • POS - adjust the Place of Service code
  • Proc - view the procedure code
  • Modifiers - adjust or add modifiers
  • 24 Shaded - enter information that should be included in the shaded part of Box 24j on a claim form
  • NDC - check the box if the information in 24 Shaded is a National Drug Code
  • Attribute To - view the provider under whose name the charge will be billed to insurance, if not the provider associated with the encounter
  • Units - adjust the units for this charge
  • Unit Charge - view the original charge per unit
  • Fee Sch Chrg - edit the charge on the fee schedule, if any, applied to this invoice
  • Billed - view the amount that will be billed to the payer
  • Tax - view the tax amount 
  • Allowed - adjust the amount allowed by the payer
  • EPSDT - view whether or not this charge will be billed as an Early Period Screening, Treatment and Diagnostic code
  • Fee Schedule - view the current fee schedule associated with the payer or patient
  • Tax Rate - add or adjust the tax rate for this item

Note: This section relates to insurance-related charges.

  • Prim. Payer - view the primary payer selected for this charge
  • Resp. - choose the code that best describes the type of patient responsibility
  • Prim. Owed - adjust the amount the primary payer owes
  • Prim. Paid  - view the amount that has already been allocated to this charge
  • Pymt ID(s) - view the payment ID(s) associated with any payment that has been made
  • Add Row - add a new row to enter more information about patient responsibility
  • COB - check to indicate that there is a Secondary Payer responsibility (Note: checking this box opens up an additional section of this charge; see below)
  • Pt. Resp. Dropdown  - code used to denote the type of Patient responsibility
  • Pt. Owed - used to indicate how much the Patient owes
  • Pymt ID(s) - will display the Payment ID associated with this charge
  • Adjustment - choose the adjustment code from the dropdown 
  • Amount - enter the amount of the adjustment 
  • Add Row - add another row if other adjustment information is needed
  • Balance - view the balance of the charge

Note: The numbers entered into the Prim. Owed, Owed (secondary), Pt. Owed, and (adjustment) Amount boxes should total the amount entered in the Allowed box in order to balance the ledger line.

5.  If you checked COB, a new section of the charge will open. In this section, you will be able to edit information related to a secondary payer.

  • Secondary Plan - choose the secondary plan from the dropdown
  • Type - choose the code that best describes the secondary plan
  • Owed - enter the amount that the secondary payer owes (Note: you may have to adjust other boxes to balance the charge if you enter anything into this box)
  • Paid - view how much the secondary payer has already paid
  • Pymt ID(s) - view the payment ID(s) associated with any payment that has been made

6. Once edits are made, select SAVE (in blue).

Billed Status

You can also update or change any status manually. The billed status shows:

  • PT - the patient has been billed
  • P - the Primary Insurance has been billed
  • S - the Secondary Insurance has been billed

Hold Status

You can manually change the Hold Status.

Unpost Charge

Once an invoice has been created for a posted charge, you will need to remove the charge from the invoice in order to unpost the charge.  Before removing a charge from an invoice, verify the charge is not associated with a claim or draft.

Note: If the Encounter has been signed, you will need to reopen the encounter to remove the charge.

1. Click the right down arrow to open the invoice, or click on the line to open the invoice details.

2. Use the checkboxes to select the charge(s) that you would like to remove. Click the action menu (blue ellipsis).

3. Select Remove From Invoice. 

4. Select YES to continue.

5. The charges have been removed from the invoice. You can see that there is no longer an invoice number associated with these charges.

6. Click the checkbox next to the line that is not invoiced. Click the action menu (ellipsis) and select View Charges.

7. Now check the box for the charge(s) that you would like to Unpost. 

NOTE: Unpost is not available if the chart note is signed.  You will need to return to the Encounter and reopen the note.

8. When you Unpost the charge, you will receive a prompt acknowledging the charge(s) will be removed.

9. The charge(s) have been unposted.  The charge(s) will still appear in the encounter and may be completely removed from there.

Invoices

Create Invoice

Once an invoice has been created, you can add new charges to it. If Auto Invoice is turned on, invoices are automatically created when charges are posted. However, you can create an invoice from the action menu (ellipsis) OR from Charges.

1a. Action Menu: Select the checkbox(es) next to the charge(s) you would like to manage, then click the action menu (ellipsis). Select Create Invoice. This will immediately create the invoice, which will then be visible to you in the charge line.

OR

1b. Expand the Charge Details: Select the charge you would like to manage in order to see the charge details. Once the charge line expands, click the right arrow.

2. From the Charge details page, select the charge, then click Create Invoice (in blue).

Add Charge To Invoice

From the Charges page, you can add a charge to an existing invoice. Note: To add a charge to an invoice, the patient must have an existing invoice in their ledger.

1. Select the checkbox for the charge you wish to add to an invoice. 

2.  Click the action menu (ellipsis), then select Add To Invoice.

3. From Add to Invoice, select the desired invoice, followed by ADD (in blue).

Note: Only invoices whose Guarantor, Case, Payers, and Plans match those of the charge(s) will be available for you to choose. If you are unable to select a matching invoice, review the charge billing data and edit the details, if necessary, to ensure the information matches that of the invoice.

Remove From Invoice

Charges can be removed from an invoice if they have been Auto Invoiced by mistake or if you wish to move them to a new invoice to be billed separately. Verify that charges you want to remove are not on a claim or part of a claim draft.

You can remove a charge from the action menu (ellipsis) or from Charges.

1a. Select the checkbox next to each charge you’d like to remove from the invoice. Click the action menu (ellipsis), the select Remove From Invoice.

OR

1b. Expand the charge line by clicking on the down arrow. Once the charge details are expanded, click the right arrow to open the Charges screen. In the Charges screen, click the checkboxes next to the charges you’d like to remove from the invoice, then select Remove From Invoice.

2. In the Remove From Invoice dialog box, click YES (in blue) to confirm.

Allocate Payment

In order to allocate a payment, you must add the payment into the system in Payments first. You can allocate a payment from the action menu (ellipsis) or from Charges. If you allocate a payment from this area of the system, you’ll be able to see all payments that can be applied to the charge.

1a. Select the checkbox for the charge to which you wish to allocate a payment. Click the action menu (ellipsis), then choose Allocate Payment.

OR

1b. Select the line of the charge. Once the charge details are expanded, click the right arrow.

2. In the Charges screen, click the checkboxes of the charges to which you would like to allocate a payment. Then select Allocate Payment (in blue).

3. On the Allocate Payments screen, any available payments will be displayed at the top. If there are multiple unallocated payments, select the payment you wish to allocate. Note: Only payments that match the charge criteria will be available to allocate. Charge criteria include responsibility type (patient or payer) and an available amount to allocate. In this example, we will allocate a patient payment to a charge that is the patient’s responsibility.

4. Scroll to the right at the bottom of the screen to find the section that corresponds to the payer of the payment. For our example, we’ll scroll to the Patient Responsibility section, since the patient made the payment.

5.  Reference the Available amount in the payment details section to be sure of the amount you have available to allocate. Then update the Allocated column in the Outstanding Charges section with the amount you wish to allocate. Note: Charges can be partially or fully allocated. 

6.  Select ALLOCATE (in blue).

7.  The payment details will update. Once the charge is balanced, it will be removed from the slideout.

Update Charge Responsibility

You can change the entity responsible for the charge to any one of three options: Self-Pay, Care Package, or Insurance. In this area, you can also update the Guarantor, Secondary plan, and Case Note: To update any of this information, care packages, insurance plans, guarantors, and cases must first be added to the patient’s profile. Click to learn more about adding care packages, insurance, a guarantor, and cases.

You can update this information on charges from the action menu (ellipsis) or the charge line.

1a. Select the checkbox for the charge you wish to update. Then click the action menu (ellipsis) and select View Charge.

OR

1b. Click the line of the charge you wish to update. Once the link expands, click the right arrow.

2.  On the Charges page, click the Edit icon at the top of the screen.

3.  Edit the information. From here, you can edit:

  • Bill Type – choose from Self Pay, Care Package, or Insurance (Self Pay is the default if nothing else was selected at patient check in)
  • Payer – view auto-populated information once Payer has been selected (below)
  • Payer – select the patient’s primary insurance payer
  • Secondary – select any other insurance payer associated with this charge
  • Guarantor – select a guarantor (Self is the default)
  • Case – select a case associated with this encounter
  • Authorization – select any authorization associated with the case

Note:  Secondary insurance information added or changed here will update boxes 9a, 9d, and 11d on the claim form. See Patient Ledger - Update Charge Responsibility for more information.

 

4.  Select SAVE (in blue).
