Check In

An effective Check In process is critical for the success of your practice. It's important to complete check in for all patient types, including walk-ins, to ensure that the billing details are set correctly before encounter charges are posted. Payments collected during check in will automatically be allocated to posted encounter charges. 

Check In A Patient

1.  From Scheduling, select the colored patient appointment card you wish to Check In.

2.  From the Check In slideout, select Check In (in blue) on the top right corner.

3.  Verify all appointment information is correct. If not, adjust the highlighted information. 

Note: Cases & Authorizations must first be added to the Patient Record. 

3.  Select the Billing Details, which include:

  • Self Pay - payment falls under the responsibility of the patient, or designated Guarantor
  • Care Package - allows the user to choose (or add) a care package/subscription associated with the encounter
  • Insurance - payment falls under the responsibility of the patients elected insurance payer

Note:  To update the patient's guarantor, select the Guarantor dropdown. To elect a new one, select Add Patient Guarantor (in blue).

Make A Patient Payment 

4.  Select Make Payment (in blue). 

5.  From the Add Patient Payment slideout, fill in the fields below:

  • Payment Type - select a payment code related to patient or payer payments
  • Amount - payment amount of the payment type selected
  • Payment Method - includes cash, check, debit, or credit card 
  • Transaction Date 
  • Note - information entered here will be visible when viewing this payment from the patient's Ledger.

Note:  Payment Types first need to be created in Billing Codes. Click here to learn more about Billing Codes.

6.  Once the payment fields are completed, select Post (in blue). 

7.  Once a payment has successfully posted, a green success banner will display, and the user will be presented with the Posted Payment details, allowing them to perform various actions including:

  • Void - allows the user to quickly void the payment
  • Allocate Payment - allows the user to Allocate that payment toward a specific charge
  • Print Receipt - allows the user to print a PDF receipt

8.  Select Close to return to the Check In slideout.

9.  To complete the check in process, select Complete Check In (in blue).

10.  Once the patient has successfully been Checked In, a green success banner will display and the Appointment Status will update. To return to Scheduling, select the X (top right corner). 

Unable to Add Case

If you try to Check In a patient and are unable to add Case, check to see if an encounter has been created for this appointment and if the charges were posted to it. 

2.  The easiest place to check is the Patient Ledger under Activity.  The charges for this encounter have been posted.  

3. To add the case, select the ellipses (in blue) and select View Charges.

4. Now from Update Charges, select the icon to enable the selections in this row and select the Case from the dropdown menu.
