Patient Record - Cases

From the Cases tab of the patient record, you can create a case to track and manage treatments or episodes outside of routine care, as well as create and manage authorizations for care. 

Add A Case

1.  Select Add Case (in blue).

2.  From the Add Case slideout, fill out the sections below:

  • Name - the name of the case
  • Care Type - choose the care type 
    • Wellness Care
    • Employment Injury - requires an injury date and state entry 
    • Auto Accident - requires an injury date and state entry 
    • Other Accident - requires an injury date and state entry 
    • Short Term Episode (Acute)
    • Long Term Episode (Chronic) 
  • Guarantor - select a guarantor for this case as a reference for billing activities
  • Claim # - claim number this case relates to

3.  If there is not a payer, leave this field set to Self.  To add a payer, you will need to click the "x."  The box will turn red with the text Primary Payer. Now you can select the Primary Insurance, which will auto populate the Primary Payer box, and add a Secondary Payer if applicable.

  • Primary Payer - set to "Self" indicating the responsibility falls on the patient
  • Primary Insurance - if this case is covered by an insurance plan, make that selection here
  • Choose Secondary Insurance - add a second insurance payer here

Note: All required information to add a case is found in the above. The next sections are all additional information that could be necessary for certain types of claims.

Dates

Add any Dates that are relevant to this case. Depending on the care type and the payer, certain dates may be required for billing even if they are not required fields here. 

  • Date of Onset - select between an injury date or gradual onset 
    • Initial Treatment Date 
    • Last Seen Date
    • Accident Date
    • Accident State
  • Unable to Work - indicate the work status of the patient 
    • Disability Begin Date
    • Disability End Date
  • Hospitalized -  indicate the patient was hospitalized for this given case
    • Hospital Admission Date
    • Hospital Discharge Date

Additional Dates

  • Date Type - choose the date type
    • 453 - Acute Manifestation of Chronic Conditions 
    • 455 - Last X-Ray
    • 471 - Prescription
    • 90 - Report Start (Assumed Care Date)
    • 91 - Report End (Relinquished Care Date)
    • 444 - First Visit or Consultation 
  • Date - date of the selected Date Type 

Additional Information

Include any additional information related to this case.

  • Referring Provider - used to indicate the referring provider for this case and treatment 
  • Ordering Provider - used to indicate the ordering provider for this case and treatment 
  • Attorney Name - used to indicate the attorney affiliated with this case 
  • Attorney Phone
  • Search Organization - used to select an organization affiliated with this case

Note: Organizations must first be created in Settings. Click here to learn more about Organizations. 

  • Note - enter any additional details related to this case 

6.  Select SAVE CASE (in blue).

Note: Once a patient case is created, it can be linked to an encounter during Check In. Click here to learn more about Check In.

Update Case

1. Click anywhere in the row of the Case you wish to update.

2.  From the Update Case slideout, update the CASE INFO and SAVE CASE (in blue).

Add Authorization

1.  Select a Case you have already created. Note: Only one authorization can be added to each case. 

2. From the Authorizations tab, select Add Authorization (in blue).

3.  From the ADD AUTHORIZATION slideout, update the authorization details highlighted below. 

4. If you Add Diagnosis (in blue) and/or Add Procedure (in blue) now, when they are added to an encounter, they will accrue towards the total authorized.

  • The total allowed, must equal the Total Authorized (We authorized 9.  The total allowed procedures must also equal 9.)

5. Once the information has been filled out and SAVED, the authorization will be listed for that case.

Note: Once an authorization is created, it can be linked to an encounter. Click here to learn more about Appointments.

Using Authorizations

1. When you Check In a patient, it is important to select the case and authorization. Authorization dates will need to be within the dates of service to count toward the authorization.

2. Proceed normally with the encounter. Once the encounter is complete, POST the charges.

3. If not all the charges posted are covered by the authorization, a notification will display.  If you choose to Yes, the charges not covered will become the patient's responsibility.

4. Be sure to add the Case and Authorization for all appointments to which they apply. ChiroTouch will keep track as you use them. You may go back into Update Authorization to see which invoices contain the applicable procedures and see how many authorizations remain.

5. Once this Authorization has been used up, a warning message will display on the encounter.

Note: Remember that only one authorization can be added to each case. Once all procedures/services have been rendered, you will need a new authorization.

View Linked Case Records

1.  Select the Case whose records you wish to view.

2.  From the Update Case slideout, select the Linked Records tab to see the linked Encounters for that case.

Change Date of Onset (Billing)

When a case has been added, you might need to modify the Date of Current Illness  (FL14) on a claim form.  The default for this field is the Case date unless you have changed the Date of Onset in the encounter Diagnoses.

1.  To modify the claim field, go to the claim form for the patient.  Use the icon in the top right corner of field to open the dialog box.

2.  A pop-up will display the dates available for this field.

3. Select the Case-"Date of Onset" if that's what's preferred.
