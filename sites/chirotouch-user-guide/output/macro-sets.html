Macro Sets

Macro Sets allows the practice to customize the built-in macro sets (EasyTouch and BulletTouch) and create their own custom macro sets from scratch. Custom macro sets can include all of the same features as the built-in macro sets, such as Menu Buttons, Menu Link Buttons, Text Buttons, and Advanced Macros. 

Default Macro Sets

By default, the EasyTouch & BulletTouch built-in Macro Sets will display. The built-in macro sets (EasyTouch and BulletTouch) cannot be directly edited. You must first create a copy to edit these, which can then be edited. Pre-set macro sets are shown as authored by IPS ChiroTouch. 

  • EasyTouch – presents essential patient detail sufficient for most cash practices
  • BulletTouch Evaluation - provides more detail than the EasyTouch macro sets specific for evaluations and is better suited for the requirements of 3rd party payers
  • BulletTouch Routine Visit - provides more detail than the EasyTouch macro sets specific for routine visits and is better suited for the requirements of 3rd party payers

Customize Macro Sets from a Copy

1. Select Macro Sets from Practice Settings, followed by the horizontal ellipsis (in blue), followed by Copy Macro Set.

2. Click anywhere in the copy of the new macro set to open it. 

3.  From the Update Macro Set, edit the Set Name.

4.  Starting with the Subjective section, select the desired Menu Button (blue button) or Text Button (green button) to start editing.

  • Menu Buttons (blue)  – menu buttons contain an additional page of macro text buttons. This is used to organize the macro set and to give more room for additional macro buttons

  • Text Buttons (green)  – text buttons will insert formatted text or advanced macros into the patient’s chart note. Advanced macros include text replacement and question-based macros

  • Note: These buttons can be dragged and placed into the desired order.

5. Select a menu or text button to customize that button.

6. Blue text indicates Advanced Macros. When used in charting, these will display as either question and answer dialogs or commands that will automatically insert demographic, patient health information, and visit information into the chart note. These can be altered to suit your needs.

7. If you click the blue text in an already created macro button, a pop-up window will appear to EDIT that particular macro.

8.  Once you are finished editing, click DONE.

9. SAVE your personalized macro set.

Add An Advanced Macro

If you have a particular macro that you would like that is not a current option in your macro set, you can create your own Advanced Macro.

1. From the Update Macro Set screen, you can see a list of available macros and insert a new Advanced Macro (in blue).

2. Select the Filter Type to display a list of available advanced macros from which to select.

Available Advanced Macros

  • Ask - presents a text input used to record answers that require text entry
  • Choose Any - presents a list of choices used to record answers for questions that can have multiple answers
  • Choose One - presents a list of choices used to record answers for questions that require a single answer
  • Date - presents a date picker used to record answers for questions that require a date
  • Number - presents a number pad used to record answers for questions that require a numerical value
  • Region - presents a body image used to record answers that require both region selection(s)
  • Spine - presents a spine image used to record answers that require vertebrae selections
  • Address - inserts the patient’s address into the chart note based on the Address 1, Address 2, City, State, and Zip Code fields in the patient’s profile
  • Age - inserts the patient’s current age into the chart note based on the Date of Birth field in the patient’s profile
  • BIRTH_DATE - inserts the patient’s birth date into the chart notes based on the Date of Birth field in the patient’s profile
  • CURR_DATE - inserts the current date into the chart note
  • CURR_DX - inserts the patient’s diagnosis for the current encounter into the chart note
  • CURR_LISTINGS - inserts the patient’s current listings into the chart note
  • CURR_TIME - inserts the current time into the chart note
  • CURR_TX - inserts the patient’s current treatment plan into the chart note
  • CURR_USER - inserts the first and last name of the current user into the chart note
  • Email - inserts the patient’s email address into the chart note based on the Email field in the patient’s profile
  • EMP_STATUS - inserts the patient’s employment status into the chart note based on the Employment States field in the patient’s profile
  • FIRST_NAME - inserts the patient’s first name into the chart note based on the First Name field in the patient’s profile
  • He/She - inserts the appropriate word into the chart note based on the Sex field in the patient’s profile
  • Him/Her - inserts the appropriate word into the chart note based on the Sex field in the patient’s profile
  • His/Her - inserts the appropriate word into the chart note based on the Sex field in the patient’s profile
  • INJURY_DATE - inserts the patient’s date of current illness for the current encounter into the chart note
  • LAST_NAME - inserts the patient’s last name into the chart note based on the Last Name field in the patient’s profile
  • MAR_STATUS - inserts the patient’s relationship status into the chart note based on the Relationship Status field in the patient’s profile
  • Mr/Mrs - inserts the appropriate word into the chart note based on the Sex field in the patient’s profile
  • PHONE - inserts the patient’s phone number into the chart note based on the Phone Number field in the patient’s profile
  • SEX - inserts the patient’s sex into the chart note based on the Sex field in the patient’s profile
 
3. Once an Advanced Macro is selected, update the details and choices accordingly. To link pre-created Choice Templates, select Link Choice Template (in blue).
  • Choice Templates - allows users to apply choice options based on the Advanced Macro selected quickly. These can be used with Choose One, Choose Any, and Region Advanced Macros. Updating a choice template will update all questions associated with that choice template. You can unlink a question from a choice template so that the question can be edited independently.
  • Save Choice Templates – allows users to save the newly created choice template for future use.

4. Repeat the above steps for the remaining sections (Subjective, Objective, Assessment, Plan) followed above SAVE (in blue).

5.  The newly created macro set will display.

Create A Custom Macro Set

1. Select Macro Sets from Practice Settings, followed by Add New Macro Set (in blue).

2. Update the title and click anywhere in the empty cell. 

3. From the Select Macro Button Type slideout, select the desired button type.

 4. Once selected, give your button a Text Name or select Advanced Macro (in blue). Once an Advanced Macro is selected, it will display as blue, clickable text. Edit the Advanced Macro (see the above Advanced Macro section).

5.  Repeat the steps above for the remaining sections (Subjective, Objective, Assessment, Plan) followed by SAVE (in blue).

6.  The newly created Macro Set will display along with your other macro sets.

Macro Set Order

The Macro Set order displayed here will determine how they display when charting an encounter. Select the drag icon (in blue) to drag and drop each macro set into the desired order.

Macro Set Action Menu

The Macro Set action menu (ellipsis) will display a list of actions available for the user to perform.

  • Copy Macro Set – allows users to copy existing macro set
  • Edit Choice Templates – allows users to update a choice template and questions associated with that choice template. You can unlink a question from a choice template so that the question can be edited independently. 
  • Inactivate Macro Sets – allows users to make macro sets inactive

Export Custom Macro Set

Custom macro sets can be exported from ChiroTouch Core or Advanced for another ChiroTouch Core of Advanced user.

1.  From the ellipses, select Export Macro Set.

2.  The .ct (ChiroTouch custom macro) file will save to your drive.

 Import Custom Macro Set

Custom macro sets can be imported to ChiroTouch Core or Advanced from another ChiroTouch Core of Advanced user.

1.  Select Import Macro Set.

2.  Drag and drop or select the .ct (ChiroTouch custom macro) file.

3. The imported macro set will now show in your macro sets.
