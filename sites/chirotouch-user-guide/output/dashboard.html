Dashboard 

The Dashboard displays pre-built, practice-related reports which provide data to see trends and find growth opportunities within your practice. You can manage and export reports and track the practice's performance. The Dashboard is divided into Daily, Appointments & Encounters, Billing, and Audit Log.

The Dashboard data generally updates every hour.  But due to internet speeds and the amount of data, updates might take up to twenty-four hours.

In the upper right corner, you'll notice icons for Export and Refresh.

View Details

1. Use the icon (upper right corner of each report) to expand and collapse each report to view specific details and change parameters. Parameters can be changed for viewing but cannot be saved.

2.   Once a report has been expanded, the details can be changed for Group, Granularity, Sub-Group, Time Range, and Date Range. Use Export to save a copy to share with others.

Group 

The system default for this graph is Start Time UTC.  And although you can change these details, they may not offer the best data for spotting trends.

Granularity

The system default shown here is Month.

Sub-Group

The system default shown here is for Appointment Type.

Time Range

This default time range is set from Start of Year to End of Month.

2. If you click Presets, a list of options appears to change the level of detail.

 

Date Range

Use the slider bars to adjust the visible date range.

Export

Use the ellipses (upper right corner) to open the menu for exporting data.

Individual Bar Detail

1. If you hover over an individual bar on the graph, you can see the details for it.

2. If you click on the bar, a DETAILS button will appear.

3.  Click on the DETAILS button, and the raw data will come up for that particular bar. Use the slider bars along the side and bottom to view the entire report. 

Action Menu

When you click on the action menu (ellipses) for a report on the home screen, you'll see a menu for each report different from the action menu you'll see with the expanded report.

This action menu has quick ways to view and change:

  • Filters - add preferences to get results that suit your needs
  • Time Bar - change the date used to filter results
  • Sort & Limit - set how results are sorted & set limits on how many results to show at a time
  • Rulers - change the scale of your metrics, add gridlines, show reference lines to help achieve goals
  • Info - change the title of your chart and add a description
  • Export - screenshot (PNG) or export as a PDF or CSV file

The Daily dashboard will be the default tab that opens when navigating to the Dashboard area with seven new visuals that focus on the current day.
