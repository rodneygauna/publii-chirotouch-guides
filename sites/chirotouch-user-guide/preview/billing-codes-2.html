Billing Codes 

In the Billing Codes, users can add billing-related codes including Write Offs/Adjustments, Payment Types, Charges, and Void/Refunds. Below you will learn how to add and manage all billing code types.

Create Write Offs/Adjustments

Create and manage reasons why you would write off or adjust differences between the practice's fee and the received payment. Entries must be a unique combination of Group and Code values.

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Billing Codes.

4.  Select Add Adjustment Code (in blue).

5.  In the newly added row, update the following information:

  • Type - used to determine if this code should be used for the following charge types
    • Patient
    • Insurance
    • Both
  • Group - the group the code relates to
    • CO
    • OA
    • PI
    • PR 
  • Code - code of adjustment type
  • Description - used to describe the code entered

6.  Once complete, select SAVE.

Create Payment Type Codes

Create and manage codes related to patient & insurance responsibility payments. When you collect payment the system will present these types to then capture related detail.

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Billing Codes.

4.  Select Payment Types.

5.  Select Add Payment Type Code.

6.  In the newly added row, update the following information:

  • Source - used to determine if this code should be available for selection when making payments from the following payment types
    • Patient
    • Insurance
    • Both
  • Code - abbreviation of the code displayed when making a payment
  • Description - used to describe the abbreviated code entered above

Note: Click here to learn more about making a payment.

Note: Require procedure at time of collection - If the charge requires a specific procedure to be identified at time of collection, enable "Require procedure at time of collection". This would require the user to allocate the payment to a specific procedure before the payment is posted.

7.  Once complete, select SAVE.

Create Charges

Create and manage codes related to charges outside of performed encounter procedures. This includes goods, packages, or fees.  Especially helpful when you need to add a charge where the value might change such as a massage package or wellness package. When you add a charge to a patient ledger the system will present these types to then capture related detail.

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Billing Codes.

4.  Select Charges.

5.  Select Add Charge Code.

6.  In the newly added row, update the following information:

  • Type - used to determine if this code should be available when adding a charge for the following charge types
    • Purchase
    • Fee
  • Code - abbreviation of the code displayed when adding a charge
  • Description - used to describe the abbreviated code entered above

Note: Auto-generate invoice - If you want the system to generate an invoice upon posting that charge, check the box for an Auto-generate invoice for that charge.

Require payment at time of charge - If you require that the user collect payment at time of charge to post the charge, enable Require Payment at time of charge.

7.  Once complete, select SAVE.

Create Void/Refund Codes

Create and manage codes related to voids and refunds of payments. When users indicate a payment should be refunded or voided they must select one of these types.

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Billing Codes.

4.  Select Void/Refunds.

5.  Select Add Void/Refund Code.

6.  In the newly added row, update the following information:

  • Code - abbreviation of the code displayed when voiding or refunding
  • Description - used to describe the abbreviated code entered above

7.  Once complete, select SAVE.
