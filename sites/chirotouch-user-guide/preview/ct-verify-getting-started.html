CT Verify - Getting Started

Congratulations!  You've decided to subscribe to CT Verify. Once our support team activates CT Verify on your account, a Real-Time Eligibility tab will be available for each payer. 

If you are already set up with Change Healthcare Clearinghouse and your settings are saved in ChiroTouch, you can now set up your Payer Plans.

Set Up Payer Plans  

In Payer Plan Maintenance, select the payer you'd like to set up. In the Update Payer Plan slideout, select the new tab for Real-Time Eligibility.  

You'll need to enable Real-Time Eligibility for each payer. By default, Real-Time Eligibility is disabled.

You will need to fill in a Real-Time Eligibility ID or the Real-Time Eligibility Portal ID.

Get Real-Time Eligibility Information 

To get this information for each payer, you'll need to be logged in to the Change Healthcare ConnectCenter.  Under Payer Tools, select Payer Search.

Use Payer ID to Search

If you know the Payer ID, that is the easiest way to search. From the Payer Search, you type in the Payer ID, select Eligibility and Professional, then SEARCH.  The results will be returned for only that particular payer.

The PAYER ID search returned results for a single-payer, all with the same Realtime Payer ID. 

Search All Payers

If you want to search the entire list of payers, from Payer Search, you can select Eligibility and Professional, then SEARCH.

You'll get a list of all payers. 

The two columns to look at are Realtime Payer ID and Connectivity Available. 

  • If a payer you use has a RealTime Payer ID, you can put those results for your Real-Time Eligibility ID.
  • Check the Connectivity Available column if your payer is missing a RealTime Payer ID. If it has a P in that column, follow the instructions to obtain and enter the Portal ID.

Real-Time Eligibility ID

If you use a payer with the Realtime Payer ID listed, you can update the Payer Plan.

Select Real-Time Eligibility ID and enter the Realtime Payer ID from the search results for that payer.

Portal ID

To obtain the Portal ID, click on the CPID of the payer.

A new window will open will all of that Payer Details.

Type the Eligibility Portal ID into the blank for PORTAL ID from the Payment Details page. You'll also need to put in your Portal Username and Portal Password.

NOTE: Use your Portal Username and Portal Password you have obtained for each payer.  If you don't have a username and password for a particular payer, then click View to sign-up with that payer.
