Patient Record - Clinical

The Clinical tab from within the patient's profile consists of 4 subsections: Chart, Problem List, Treatment Plan, and Encounters. Below you learn more about each subsection and the various functions related to each.

Chart Tab

In the Chart section, users have the ability to view, add, and edit:

  • Active Problems
  • Recent Encounters
  • Treatment Plans
  • Encounters

Active Problems 

Active Problems will display a list of the patient's current diagnosis, description, and type. From here, users also have the ability to quickly add a problem. 

Add Problem

1.  Select Add Problem (in blue).

2.  Users will be routed to the Problem List section of the Patient Record.

3.  Enter the problem in the Search Diagnoses.

Note: Problems can be searched using any combination of key letters, words, or ICD10 code.

4.  Select Add to Problems (in blue).

5.  From the Add to Problems List, fill out the pertinent information:

  • Provider - select a provider
  • Type - labels the problem as:
    • Acute - severe and sudden in onset
    • Chronic - require ongoing medical attention or limit activities of daily living or both
    • Self-limiting - a disease process that resolves spontaneously with or without specific treatment
  • Status - status of the problem
    • Active
    • Inactive
    • Resolved
  • Onset Date - date this problem started
  • Resolution Date - date this problem resolved
    Note: This field will be disabled until the problem status is set to "Resolved" and the "Onset Date" is populated.
  • Note - add a note to include more detail about the added problem 

6.  Select ADD (in blue).

7.  Once added, select SAVE PROBLEMS (in blue).

Note: Problems added while Charting Encounters will also display here. 

Problem List Tab

In the Problem List section, users have the ability to view, add, update and delete problems. 

1.  Click the problem row you wish to update.

Update

2a.  Click anywhere on the Problem to open the Update Problem slideout.  Update the desired information, then select UPDATE (in blue).

Delete

3a.  Click anywhere on the Problem to open the Update Problem slideout. Then select DELETE.

3b. The Remove Problem box will appear, select YES (in blue) to confirm.

Resolve

4a.  From the Update Problems, update the status to Resolved and add a Resolution Date.  Then click UPDATE (in blue).

Treatment Plan Tab

The Treatment Plan will display treatment plans for the given patient showing current and completed phases within their overall plan. 

1.  Click within the Treatment Plan section.

2.  Users will be routed to the Treatment Plan section of the Patient Record.

3.  Select Add Treatment Plan (in blue).

4.  Choose a Start Date.

5.  Enter the Phase Details in the highlighted section below which include:

  • Number of Treatments
  • Frequency
    • Days
    • Weeks
    • Months
  • Duration
    • Days
    • Weeks
    • Months
  • Note: This example shows 3 treatments every 1 week for 6 weeks.

6.  Select Add Phase (in blue above).  The phase details entered above will calculate to display the following information:

  • Apt - number of appointments this phase includes
  • Com - number of appointments currently completed in this phase
  • Sch -number of scheduled appointments remaining in this phase
  • TBS - number of appointments yet to be scheduled for this phase

8.  For a Multi-Phase treatment plan, select Add a Phase (in blue).

Note: Appointments will not be scheduled automatically from within this section. Click here to learn more about Scheduling. 

9.  Select Save Treatment Plan (in blue).

10.  Once saved, the treatment plan will display in the patient's chart.

Encounters Tab

Recent Encounters will display a list of encounters for the given patient. From here, users can view, edit, sign, and print encounters. 

View, Edit, Sign, And Print Encounters

1.  Click on the recent encounter you wish to perform an the action.

2.  The selected Encounter Summary displays (in blue) along with performable actions related to that encounter.

Print Multiple Encounters

1.  Select the encounters that you would like to print.

2. Then select Print Selected Encounters.

3.  The selected encounters will open in a new window as a PDF.  

Note: There is a maximum of 25 encounters per PDF.

Note: Users Permissions & Rights determine who can perform the above actions. Click here to learn more about Permissions & Rights. 

Printed Encounters

Printed Encounters contains the Patient's full name, DOB (Date of Birth), MRN, and DOS (Date of Service) on each page for easy identification.
