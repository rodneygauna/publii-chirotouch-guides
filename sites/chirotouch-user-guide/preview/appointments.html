Appointments  

Scheduling Appointments is quick, easy, and available from various areas throughout the application. Below you will learn how to add, reschedule and update appointments.

Add New Appointment

Appointment Types must first be created in Settings. Click here to learn how to create Appointment Types.

1a. From Scheduling, select ADD (in blue) followed by Add New Appointment (top right corner).

Or

1b. From Scheduling, select an open time block.

2.  From the New Appointment slideout, use the Search Patients search box, to find the desired patient using the options below:

  • First, Middle, Last & Preferred Name
  • Date of Birth
  • Medical Record Number (MRN)
  • Phone Number

3.  Select and update the following options:

  • Provider - select from dropdown (required)
  • Appointment Type - select from dropdown (required)
  • Case - optional
  • Authorization - optional
  • By default, the One-Time Appointment card will be selected. Click here to learn more about the Recurring Series.

Note: Cases & Authorization must first be added to the Patient Record. 

6.  Select by:

  • Date - allows users to search for open time slots based on the desired date
  • Time - allows users to search for open time slots based on the desired time range
  • Override - allows users to schedule appointments regardless of provider availability and open time slots
  • Choose the Duration in Hours and Minutes.

8.  Once updates are made, select Schedule (in blue).

Note: Available appointments per provider can be filtered using the Select by Date, Time, and Override option. These options filter appointment availability based on the selected Providers Availability set up in Provider Settings. Click here to learn more about  Provider Availability.  

Reschedule Appointment

Rescheduling an appointment allows the user to change appointment details such as provider, appointment type, date, and duration. 

1a From Scheduling, click on the colored Patient Appointment Card you wish to reschedule.

Or

1b. From Scheduling, drag the colored Patient Appointment Card you wish to reschedule to an open time block.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Reschedule.

3.  From the Reschedule Appointment slideout, adjust the appointment details.

5.  Select a Reschedule Reason from the dropdown and include a Rescheduled Note if necessary.  Note: If Reschedule Reasons were first created in Settings, follow the below steps. 

6.  Select Reschedule (in blue). 

Confirm Appointment

If you Confirm an appointment, you'll have a visual cue on the schedule that lets you know you've communicated with your patient.

1.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Confirm.

2.  Confirm Appointment by clicking YES.

3.  You'll see the appointment has been Confirmed in the Update Appointment slideout.

4.  You'll also notice an icon for the patient that indicates that this appointment has been confirmed in both Scheduling and Charting.

Update Appointment

Updating an appointment allows the user to update appointment details such as the appointment case, along with canceling and deleting appointments. 

Edit Appointment

Edit Appointment allows you to update the patient's case and notes. 

1.  From Scheduling, click on the colored Patient Appointment Card you wish to update.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Edit.

3.  Make updates and SAVE (in blue).

Cancel Appointment

Users can cancel appointments due to patient-related reasons. 

1.  From Scheduling, click on the colored Patient Appointment Card you wish to cancel.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Cancel.

3.  From the Cancel Appointment dialog, select a Cancel Reason from the dropdown and include a Cancel Note if necessary.

4.  Select Yes (in blue). 

Note: Cancel Reasons must first be created in Settings. Click here to learn how to create Cancel Reasons.

Delete Appointment

Users can delete accidentally created appointments. 

1.  From Scheduling, click on the colored Patient Appointment Card you wish to delete.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Delete.

3.  From the Delete Appointment dialog, select YES (in blue).
