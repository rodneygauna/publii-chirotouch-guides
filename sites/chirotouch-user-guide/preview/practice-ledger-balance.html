Practice Ledger - Balance

The Balance card displays both payer and patient balances for your entire practice. Only open charges will display, which allows you to see at a glance what needs to be balanced. In addition to balancing open charges, you have the ability to create patient statements and superbills from the Balance card.

Use the filters to narrow your search. The Date of Service automatically defaults to the last 30 days.

Summary Tab

The Summary tab displays a Charges Balance table that displays information about outstanding patient and payer balances.

Update Charges

Billed Status

You can also update or change the status manually. The billed status shows:

  • PT - the patient has been billed
  • P - the Primary Insurance has been billed
  • S - the Secondary Insurance has been billed

Hold Status

You can manually change the Hold Status.

Statements Tab

In the Statements tab, you can generate, preview, and track patient statements. You can filter statements in this screen using the following fields:

  • Patient - search for a particular patient
  • Statement ID - enter the statement number if it's known
  • Date - add a date range
  • Guarantor - enter a guarantor
  • Balance - add a minimum and/or maximum balance amount

Create Statements

1. From the Balance card, select Statements, followed by Generate New Statement (in blue).

2. To generate a statement for an individual patient, search for the patient, then choose the appropriate options to generate the statement:

  • Search for a Patient – Select the patient(s) that you want to include
  • Date Range – select a date range for the charges, if desired; if Include Previous Balance is selected, the date range will be required
  • Payers - select which payers to view from the dropdown
  • Providers – select one or more providers
  • Include Previous Balance – select to include the previous balance carried over from the last statement
  • Include zero dollar charges - select to include charges that equal $0
  • Include Aging Table – select to include an aging table of charges in 30, 60, and 90-day increments

3. Once you Generate a statement, it will open in a new browser window.  From here you can view, print and save. 

4. You can return to the statement later and click on the line in the table to reopen it.

Batch Generate Patient Statements

1. From the Balance card, select Statements, followed by Generate New Statement (in blue).

2. From the Generate New Statement dialog, click on Search for a Patient. Choose the first option of All Patients Seen. This option will only generate patient statements for patients seen with services rendered within a selected date range.

Note: A date range is required when selecting this option.

3.  Once All Patients Seen is selected, choose from the following options, followed by GENERATE (in blue).  

  • Date Range - select a date range of charges or services rendered. This field is required if the All Patients Seen option is selected or if the Include Previous Balance checkbox is selected
  • Payers - select which payers to view from the dropdown
  • Providers - select all or individual providers
  • Include Previous Balance - will include a balance forward line on the patient statement, a date range is required
  • Include zero dollar charges - select to include charges that equal $0
  • Include Aging Table - displays an aging table of charges in 30-day increments 
  • Generate - opens the Patient Statement in a new tab to print while logging in into their account

Note: Preview is only available for individual patient statements. 

4. Once Generate New Statement dialog closes, select View Batches (in blue) to view batches.

5. From the View Batches slideout, processing and completed batches will display. Once completed, select the download icon.

6. Once downloaded, the Bulk Patient Statements will open up in a new tab in your browser to print and manage.

7. Individual patient statements created out of the batch will now display within the statement table along with the associated Batch ID.

Note:  Individual patient statements not created in a batch will not have an associated Batch ID.

Group by Relationship

Statements can be generated for patients that share relationships.

1.  From Generate New Statement, select one of the patients in the relationship and then check the box for Group by Relationship.  Select the Related Patients you want to Generate a statement for.

2. A PDF will open in a new window with All Related Patients, that were selected, grouped together on one statement with their charges separated out individually.

3.  At the bottom of the group statement, you'll see the total of all charges and payments for the Related Patient Statement.

Superbills Tab

The Superbills Tab in the Balance card allows users to generate, preview, and track Patient Superbills. Below you will learn how to create Patient Superbills.

  • Patient - search for a particular patient
  • Superbill ID - enter the Superbill number if it's known
  • Payers - select the person or entity paying the bill
  • Date of Service - add a date range
  • Payers - select the person or entity paying the bill
  • Providers - select one or more providers
  • Total Charges - add a minimum and/or maximum charge amount

Create Superbills

1. From the Balance card, select Superbills, followed by Generate New Superbill (in blue).

2. You can filter statements in this screen using the following fields: 

  • Date Range - select a date range of charges or services rendered if desired 
  • All Payers - choose from the Payers associated with the patient's charges or choose Self Pay 
  • Unique Superbill Per Payer - creates a unique Superbill per payer selected in the Payer dropdown
  • Providers - select all or individual Providers
  • Charges without a patient balance - if selected, includes charges without a patient balance on the Superbill
  • Include Adjustments - if selected, includes adjustments 
  • Generate - opens the Patient Superbill in a new tab to print, while logging in to their account
  • Preview - opens the Patient Superbill in a new tab to print, but does not log it into their account

3. Select GENERATE (in blue).

4. Once generated, the Patient Superbill will open up in a new tab to print, and now display within the patient's chart. 

Missing DX Pointers

If you generate a Superbill and the DX Pointers are not present on the charges, you will need to edit the charge line in the ledger.  If an encounter has been signed, the encounter will need to be reopened to add the diagnoses to the charges.

Batch Generate Superbills

1. From the Balance card, select Superbills, followed by Generate New Superbill (in blue).

2. From the Generate New Superbill dialog, select the patient search bar followed by the first option of All Patients Seen. This option will only generate patient statements for patients seen with services rendered within a selected date range. Select from the following options, followed by GENERATE (in blue).

  • Date Range - select a date range of charges or services rendered. This field is required if the All Patients Seen option is selected.
  • All Payers - choose from the Payers associated with the patient's charges 
  • Unique Superbill Per Payer - creates a unique Superbill per payer selected in the Payer dropdown
  • Providers - select all or individual Providers
  • Charges without a patient balance - if selected, includes charges without a patient balance on the Superbill
  • Include Adjustments - includes adjustments if selected 
  • Generate - opens the Patient Superbill in a new tab to print, while logging in to their account

4. Generate box will close.  Now you can View Batches (in blue).

5. From the View Batches slideout, processing and completed batches will display. Once completed, select the download icon.

6. Once downloaded, the Bulk Patient Superbills will open up in a new tab in your browser to print and manage. Individual patient statements created out of the batch will now display within the statement table along with the associated Batch ID.

Failed Batch

The process for removing failed batches for both Statements & Superbills remains the same.

1. If you get a status of Failed, you will need to remove the batch and reprocess.  Use the minus icon (in blue) to remove.

2. From the Remove Failed Batch dialog, select OK (in blue).
