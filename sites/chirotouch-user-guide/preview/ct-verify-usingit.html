CT Verify - Using It

CT Verify is our ancillary offering for Advanced ChiroTouch practices designed to provide real-time patient insurance verification. We invite you to explore all the possibilities and peace of mind that CT Verify offers. 

  • Learn how to configure your patient insurance plans using Real-Time Eligibility
  • Find out all the places CT Verify shows patient insurance status 
  • See how insurance history is stored for later use 
  • Discover how easy it is to choose between automated eligibility check and manual check 

Setting Up Patient Insurance 

Now that you've set up your payer plans with Real-Time Eligibility, it's easy to set up your patients' insurance policies. 

1. Select Add New Insurance and click Use Real-Time Eligibility.

2. Fill out the Payer and Insured ID fields to send the initial Real-Time Eligibility.

Note: Only payers that have Real-Time Eligibility set-up will be available in the Payer dropdown.

3. Once submitted, you’ll receive a status response. The plan will be added to the patient’s insurance with all data currently available from the payer.

4. If the system returns a empty red circle and a message, there is a system error with the transmission of data. Check to make sure the patient name and Insured ID are correct. Wait a few minutes and try to submit again. If the problem does not resolve, contact support.

5.  A filled-in red circle signals that when eligibility was checked, the policy came back Inactive.  This could mean a variety of things. Check the RTE History Log for a better description.

6. You can also view a description of these status messages in the RTE History Log to find out how to proceed.

Visibility

Insurance verification status will be shown throughout the program.  In most places, it will be indicated by circles and checkmarks.   There are 3 places where a Status Description may be viewed for clarification of the RTE icons:

  • Patient Insurance - RTE History Log
  • Check In
  • Check Out

Enable Auto Check 

With CT Verify, you can automatically check eligibility for all patients. 

1. Go to Billing Settings and select Clearinghouse Settings. Your Change Healthcare Submitter information should be filled in. 

2. Under Real-Time Eligibility Settings, Enable Auto-Check.

2. Use the dropdown to select the Number of days prior to the appointment you’d like to check automatically. Auto-Check runs at 1 am PST based on the number of days prior to the appointment that you've set. 

3. Set the Numbers of days before checking again. This is the threshold on which the system rechecks the insurance if Auto-Check is enabled. 

Note: This will not affect your ability to use Real-Time Eligibility to manually check availability during Check In, Check Out, or when updating a patient's insurance plan.
