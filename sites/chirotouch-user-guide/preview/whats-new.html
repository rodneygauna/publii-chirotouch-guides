What's New 

ChiroTouch is constantly working to make enhancements to better our program.  Because we frequently put out new releases, it's good to log out to ensure the software picks up the latest changes. It's also not a bad idea to do a hard refresh occasionally and clear your cookies and cache.

2022-5-2

CT Engage (Ancillary offering) - now offers you the ability to change your text and email appointment reminder and appointment confirmation messages.

2022-5-1

Dashboard - new pie graph in the Appointments and Encounters tab that displays the Payer Plan Distribution in your practice.

2022-4-22

Inventory Management - You now have the ability to keep track of items in your inventory.  

Multiple Group NPIs - Providers can now bill using the Organization NPI, Provider Individual NPI, or Provider Organization NPI for claims.  Set the provider-specific NPIs in User Maintenance for each provider.  Then select the correct Service Facility NPI (FL32a) and Organization or Individual NPI (FL33a) for each payer under Payer Plan Maintenance.

Dashboard - Two new visuals under the Billing tab.

  • Inventory by Item Name
  • Low Inventory List

Charting - Encounter summary header and footer enhancements.

  • Patient Name, Patient DOB, Patient MRN, and Date of Service are now included in the header for each printed page

  • Practice Name, Practice Address are now included in the footer for each printed page

2022-4-1

CT InForms (Ancillary offering)
  • Patient Insurance/Demographic information from completed forms will update the patient record
    • Review and approve changes before the patient record is updated
  • Review unmatched forms and assign them to the correct patient record in ChiroTouch
  • Image attachments (ex: insurance cards) write back as a separate file in the patient documents
 
Claim Status Reset
  • Now you can manually adjust the status of an electronic claim (Single or Bulk)
 
Family Care Management
  • View all patients in a related grouping from the patient record
  • Ability to share and allocate payments across all patients in a relationship
 
Claim Submission
  • Save payer's original claim reference number from ERA, so it is available for resubmission of claim
 
Online Booking
  • Removed the reschedule appointment link for walk-in appointments since it doesn't apply

2022 ICD-10-CM library has been updated
When searching for diagnosis codes, the new (total of 72,749 codes) will be shown to select from. You'll notice there are some large impacts on some commonly used diagnoses.
  • Low Back Pain
  • Spondyloarthritis
 
Deprecated codes are stripped from the Encounter Diagnosis when using "Copy Prior Encounter" and "Auto-SALT". Users will need to add the appropriate code and re-point the diagnosis to the Encounter Charges.
 
If the encounter had the deprecated code, it can still be used - along with the practice being able to add the deprecated code if the patient's Problem List (Encounter Diagnosis > Problem List) or Prior Encounter (Encounter Diagnosis > Prior Encounter) to any newly created encounters.

2021-07-01

This update contains application enhancements and issues resolved to cover both bug fixes & stability fixes. 

  • Electronic Claims Feedback Updates:
    • Pre-populate primary claim with 9, 9a, 9d, and 11d when secondary payer is set for the invoice

    • Non-Participating Payer Addition:
      • FL 12, 13 for Signature on file will always be checked by default to showing the SOF on the claim
