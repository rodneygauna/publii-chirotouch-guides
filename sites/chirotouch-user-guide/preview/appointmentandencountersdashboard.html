Appointment & Encounters Dashboard

The Appointments & Encounters tab has seven areas.

Appointments by Type

Appointments by Type shows the number and percentage of overall appointments by type of appointment booked. The default is monthly.

New Patients

New Patients shows the number of new patients versus the total patients seen. This chart is based on the number of patients Checked In and Checked Out. The default is monthly.

Appointments by Status

Appointments by Status shows the number of appointments for each status and the overall percentage of appointments by status. The default is daily.

Note: Active is for scheduled patients that have not been checked in or out.

Payer Plan Distribution

Payer Plan Distribution provides you a way to visualize the mix of payers that they bill.

Encounters by Provider

Encounters by Provider shows the number of encounters by a provider. An encounter must be started for the provider to be included in the total. The default is monthly.

Patient Recall

Patient Recall is a list of patients that have had an appointment in the past 180 days but do not have a future appointment scheduled. View each patient by patient's full name, date of birth, MRN, total appointments scheduled, last appointment date, last appointment status, email address, and phone number.

Patient Birthdays

Patient Birthdays is a quick view of all the birthdays organized by month.

Creating A Birthday List

If your office likes to send out birthday messages, the Patient Birthdays list allows you to create a patient list by month.  If you want to export your list as a CSV file, you'll first need to Ungroup the list.

1. From the ellipsis, select Visual Settings.

2. Then use the minus to remove the Birthday Month from the Edit Groups and Apply. Note: You will not be able to export a CSV file if you skip this step.

3.  Now, to create a CSV for a single month, select Filters.

4. We will need to edit the current filter.  Click inside the field for the Birthday Month to modify.

T

5.  The filter is set for months 1-12.  Let's set it for January only.  Select EQUAL.

6.  Put the month you want to filter by and Apply.

7.  Now, you can export a CSV file.  From the ellipsis, select Export, then Raw Data (CSV).
