Recurring Appointments 

Scheduling Recurring Appointments is quick, easy, and available from various areas throughout the application. Below you will learn how to add, reschedule, and update Recurring Appointments.

Schedule Appointments

1a. From Scheduling, select ADD (in blue) followed by Add New Appointment (top right corner).

Or

1b. From Scheduling, select an open time block.

2.  From the New Appointment slideout, fill out the details.

  • Use Search Patients to find the desired patient using the options below:
    • First, Middle, Last & Preferred Name
    • Date of Birth
    • Medical Record Number (MRN)
    • Phone Number
  • Select a Provider from the dropdown
  • Select an Appointment Type from the dropdown

Note: Appointment Types must first be created in Settings. Click here to learn how to create Appointment Types.

5.  Select the Recurring Series appointment card and adjust the Recurring Details including: 

  • Starts - start date of the recurring series
  • Recurs Every - Days, Weeks, Months, Years
  • On - specific days the appointment should fall on
  • Duration - duration of the recurring appointment
  • Start Time - start time of the recurring appointment
  • Ends - when the recurring series should end by Date or number of Visits 

6.  Select SCHEDULE (in blue). 

Note: Recurring appointments override provider availability and patient bookings per time slot. Click here to learn more about Provider Availability. 

Reschedule Appointments

Reschedule allows the user to change appointment details such as provider, appointment type, date, and duration. 

1a From Scheduling, click on the colored Patient Appointment Card you wish to reschedule.

OR

1b. From Scheduling, drag the colored Patient Appointment Card you wish to reschedule to an open time block.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Reschedule.

3.  From the Reschedule Appointment slideout, adjust the appointment details and select RESCHEDULE (in blue). 

5.  You may also fill out the Appointment note or select a Reschedule Reason from the dropdown, if necessary before you RESCHEDULE.

Note: Reschedule Reasons must first be created in Settings. Click here to learn how to create Reschedule Reasons.

Update Appointment

Use the Update Appointment to change details such as the appointment case, cancel an appointment, and delete an appointment. 

Edit Appointment

1.  From Scheduling, click on the colored Patient Appointment Card you wish to update.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Edit.

3.  Update the Case, Authorization, or Add an Appointment Note.  SAVE (in blue).

Cancel Appointment

1.  From Scheduling, click on the colored Patient Appointment Card you wish to cancel.

2.  From the Update Appointment slideout, select Update Appointment  (in blue) followed by Cancel.

3.  From Cancel Appointment, select a Cancel Reason from the dropdown and include a note if necessary.  Select YES (in blue).

Note: Cancel Reasons must first be created in Settings. Click here to learn how to create Cancel Reasons.

Delete Appointment

1.  From Scheduling, click on the colored Patient Appointment Card you wish to delete.

2.  From the Update Appointment slideout, select Update Appointment (in blue) followed by Delete or Delete Entire Series.

  • Delete - will only delete the selected appointment
  • Delete Entire Series - will delete the entire recurring series

Note: You can only delete a recurring series for future appointments.
