User Maintenance 

The User Maintenance section in User Settings allows you to create and update users with various role types including, providers, specialists, and staff. Only Super Users can manage staff profile information, as well as change permissions and rights.  

Every user in the practice should have their own log in credentials to help maintain HIPAA compliance.

Add New Users

1.  Select Settings.

2.  Select User Settings.
3.  Select User Maintenance.

4.  Select Add New User (in blue).

Note: To add a Provider to your account, please contact ChiroTouch support. Providers provide care to patients and have their own NPI number.  In ChiroTouch, providers can sign encounters and bill insurance for their services. 

5.  From the Add New User slideout, choose a new user type:

  • Add Specialist - user who provides care to patients, but does not have an NPI number  Note: A specialist can sign encounters, but cannot bill insurance services.
  • Add Staff - user who performs non-patient-care-related services such as scheduling, front desk, billing, etc. Note: Staff cannot sign encounters. 

6.  From the Add New User slideout, enter the desired information in each tab. Enter the first name, last name, and email address under the Profile tab.

7.  On the Specialist Identifiers tab enter the Tax ID, Primary Taxonomy, License Number or Other Identifier. 

8.  Determine which areas of the system they can access under the Permissions & Rights tab. Toggle between the gray Off position or the blue On position based on the needs of this individual user. 

Note: Note Permissions & Rights can only be adjusted from this screen by Super Users.

Permissions & Rights

To start, all users have access to view and manage basic patient information. To give users access to additional functionality, you can assign one or more permissions groups below.
  • Super User - user with Global Access who needs access to all functions in the practice, including the ability to create and manage additional practice users 
  • Patients - all users have access to view the patient roster and to manage basic patient information
  • Dashboard - users who need access to view the dashboard.
  • Scheduling - users who need access to view the schedule, create appointments for patients, and manage blocked off times
  • Charting - users who need access to view and modify patient encounters. This includes the ability to create encounters, chart encounters, and manage encounters
  • Billing - users who need access to view and manage patient ledger information, generate invoices and statements, and manage insurance claims
  • Settings - users who need access to manage and configure practice level settings

8.  Once all tabs are filled out, select SAVE STAFF (in blue).

Update Existing Users

1.  Select Settings.

2.  Select User Settings.
3.  Select User Maintenance.

4.  Select the existing user you wish to update. (Super User function only)

5.  Update the desired information in each tab.

6.  Once all tabs are updated, select SAVE STAFF (in blue).

Update Provider Information

A Super User may update the information for a Provider, but not add a new provider. 

1. From Update User Profile, you can change add a Substitute Billing Provider.

  • A provider can be selected from the list until there are no providers left to choose
  • Select a payer from the dropdown
  • Users cannot select a payer that is already checked for another provider
  • The Substitute Billing Provider's name will appear on the claim form and all their provider identifiers will be sent when submitting a claim to payers indicated in Settings
  • The rendering provider's name will still appear on the charge line in the ledger

2. On the Provider Identifiers tab update the NPI, Tax ID, Primary Taxonomy, and License Number. Note: Primary Taxonomy is a required field for claims submission. Leaving fields blank could cause billing errors.

Provider Organization NPI

If a Provider bills under a Provider Organization NPI for claims (FL32a and/or FL33a), they can add their Provider Organization number here.  It is not a required field.

Mark Existing Users As Inactive

1.  Select Settings.

2.  Select User Settings.
3.  Select User Maintenance.

4.  Select the action menu (ellipsis) for the user you wish to mark inactive, then select Disable User Access.

Note: To inactivate a provider, visit the ChiroTouch Community, then click "My Account" to manage the appropriate additional provider subscription.

5.  From the Disable User Access (Mark Inactive) dialog, select YES (in blue).

Reset Existing Users Password

1.  Select Settings.

2.  Select User Settings.
3.  Select User Maintenance.

4.  Select the action menu (ellipsis) for the user whose password need to be reset.

5.  Select Send Reset Password Email.

6.  From the Reset Password dialog, select YES (in blue).
