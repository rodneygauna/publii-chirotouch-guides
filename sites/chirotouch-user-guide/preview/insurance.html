Patient Record - Insurance

In the Insurance tab of the Patient Record, users have the ability to add and manage patient insurance and update plan information. On this page, you will learn how to add and update patient insurance. 

Add Insurance - Existing Payer 

1.  Select Add Insurance (in blue).

2. From the Add Insurance slideout, add the information:

  • Group or Plan Name - the name of the plan (could be payer name if none)
  • Payer - select the payer (For more information about adding a payer plan, go to Payer Plan Maintenance.)
  • Group ID - this is the associated with the payer, sometimes called plan ID
  • Set As - Primary, Secondary, Tertiary, Other
  • Type * 
  • Medicare Type Code 

3.  Add the information for the policyholder.

  • Insured ID or Policy # - enter the ID or Policy number of the person that holds the insurance policy (If this is not the patient, enter the number policy number for the person that holds the policy and set the policyholder relation to patient.)
  • Policyholder Relation to Patient  - defaults to Self (If the patient is not the policyholder, set the relationship.)
  • If you set the policy to other than Self, you need to fill out the name, address including 9-digit zip code, and date of birth of the policyholder for billing purposes

Note: The highlighted areas above are required to move on to the next step. However, we recommend that you enter as much information as you can in this step.

3. Select NEXT.

4. Update the Copay & Coinsurance coverage level, service types, in and out of network amounts.

5.  Select NEXT (in blue). 

6.  Update the Copay & Coinsurance coverage level, service types, in and out of network amounts.

7.  Select NEXT (in blue). 

8.  View the Summary.

Add Insurance - Without Existing Payer 

1.  Select Add Insurance (in blue).

2.  From the Add Insurance slideout, select Add New Payer (in blue).

3.  Add the information for the payer plan. For more information about adding a payer plan, go to Payer Plan Maintenance.

4.  When done adding the payer plan, click SAVE to return to the Add Insurance slideout. Then continue entering information following steps 2-8 above.

Edit Patient Insurance

1.  Select the Payer Plan you wish to edit.

2.  Select EDIT (in blue).

3.  From the Update Insurance slideout, update the plan information. 

4. Update Eligibility.

5. Add a Subscriber ID (this is patient's ID number) if the policyholder is not Self (this example is set to parent).  It will be necessary to add Subscriber ID if the patient is not the primary policyholder.

6.  Update any Coverage details such as copays, coinsurance, deductibles and max out of pocket.

7.  Select SAVE INSURANCE (in blue).

Add Coverage Details

The Coverage Details section allows users to add both covered and non-covered services to a patient’s insurance plan. When adding covered services, users can add details regarding coverage, including coverage limitations, whether or not pre-authorization is required, and copay or coinsurance amounts.

1. From the Insurance section of the Patient Record, select the insurance policy whose coverage details you wish to update. 

Note:  You can only add coverage details to an existing insurance policy.

2.  From the Insurance slideout, select the blue EDIT icon.

3. From the Update Insurance slideout, select the Coverage tab and scroll down to the bottom. 

Add Covered Services

4a. To add a covered service type, select Add Covered Services (in blue).

5a. From the Add Covered Service slideout, select Service Type.

6a.  From the Service Type dropdown, select the desired covered service type.

7a.  Once the desire Service Type is selected, fill in the service type details below: 

  • Pre-Authorization Required - used to indicate if is Authorization is required
  • Quantity Type - options include Units, Visits, Episodes, Dollar Amount
  • Quantity - required if Quantity Type is selected
  • Term -required if Quantity Type is selected
  • Remaining - remaining number of service types selected
  • As of - starting date of coverage 
  • In Network Copay Amount  
  • Out of Network Copay Amount 
  • In Network Coinsurance % 
  • Out of Network Coinsurance % 

Note: Service Type details are not required.

8a. To add another covered service, select Add Covered Service (in blue). Once all desired service types are added, select SAVE INSURANCE (in blue). 

Add Procedure Code

4b. To add a covered procedure, select Add Covered Services (in blue).

5b. From the Add Covered Service slideout, select Procedure Code.

6b.  From the Add Covered Service - Procedure Code slideout, search for the desired procedure codes. Once found, select Add to Covered Services (in blue) followed by DONE (in blue).

7b.  Once the desire procedure codes are added, fill in the service type details below:

  • Pre-Authorization Required - used to indicate if is Authorization is required
  • Quantity Type - options include Units, Visits, Episodes, Dollar Amount
  • Quantity -   required of Quantity Type is selected
  • Term - required if Quantity Type is selected
  • Remaining - remaining number of service types selected
  • As of - starting date of coverage 
  • In Network Copay Amount  
  • Out of Network Copay Amount 
  • In Network Coinsurance % 
  • Out of Network Coinsurance % 

Note: Service Type details are not required.

8b. To add another covered service, select Add Covered Service (in blue). Once all desired service types are added, select SAVE INSURANCE (in blue). 

Add Non-Covered Services

When adding non-covered services, users can list service types or procedure codes which are excluded from coverage under the patient’s plan.

1. From the Insurance section of the Patient Record, select the insurance policy whose coverage details you wish to update.

Note: You can only add coverage details to an existing insurance policy.

2. From the Insurance slideout, select the blue EDIT icon.

3. From the Update Insurance slideout, select the Coverage tab and scroll down to the bottom.

Add Non-Covered Service

4a. To add a non-covered service type, select Add Non-Covered Services (in blue).

5a. From the Add Non-Covered Service slideout, select Service Type.

6a.  From the Service Type dropdown, select the desired covered service type.

7a. To add another non-covered service type, select Add Covered Service (in blue). Once all desired service types are added, select SAVE INSURANCE (in blue).

Add Non-Covered Procedure Code

4b. To add a non-covered procedure, select Add Non-Covered Services (in blue).

5b. From the Add Non-Covered Service slideout, select Procedure Code.

6b. From the Add Non-Covered Service – Procedure Code slideout, search for the desired procedure codes and click Add to Non-Covered Services for each. Then click DONE (in blue).

Medicare Type Code

1. If you add Medicare as a secondary payer for an electronic claim, you will need to select a Medicare Type Code that will be transmitted to the Clearinghouse. 

Note: You will not see this code on the claim form. 

2.  Select a code from the dropdown.
