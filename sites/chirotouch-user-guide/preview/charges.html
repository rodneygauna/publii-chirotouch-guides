Charges

In the Charges section, users have the ability to create charges that can be used when documenting an encounter and for billing purposes. 

Add Charges

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Charges.

4.  Select ADD CHARGE (in blue).

5.  From the Add Charge slideout fill in details for each procedure with a unique description.

Note: Users have the ability to add the same procedure with a unique description and charge associated with it.  

  • Procedure Name - name of the procedure/charge which will display on encounters, and all billing related documents where applicable
  • Description - description of the procedure/charge which will display on encounters, and all billing related documents where applicable
  • Modifiers - used to provide additional information for the newly created procedure/charge
  • Amount - procedure amount billed, from zero dollars (default) and up
  • Tax Rate - tax rate used when billing the procedure/charge which includes, city and state. Click here to learn more about Tax Rates.
  • Additional Options - can be used for specific procedures/charges created
    • Durable Medical Equipment – charge is considered  durable medical equipment (DME)
    • Suppress from Claims – charge will not be billed to insurance 
    • Available for Purchase – charge can be posted outside of an encounter 
    • Timed Procedure – charge is billed by time increments (multiple units of this procedure code can be billed)
    • Bill With X-ray Date (FL 15) – charge is billed with the patient’s most recent x-ray date
    • Bill With Treatment Initiation Date (FL 15) – charge is billed with the initial treatment date    
    • EPSTD Code – charge is billed as an Early Periodic Screening, Treatment, and Diagnostic code 
    • Include Charge Description for Electronic Claims - when the box is checked, the description field will be transmitted (ie: Medicare code 97039 will have a validation error without a procedure description on claim)

6.  Select SAVE (in blue). 

Edit Charges

1.  Select Settings.

2.  Select Billing Settings.
3.  Select Charges.

4.  Select the Procedure or charge you wish to update.

5.  From the Edit Charge slideout, adjust the procedure/charge details. And SAVE when finished.

Update Charges Associated To A Payer Fee Schedule

If you update Charges here in the Billing Settings, those charges and services will update across all Payer Fee Schedules which they are associated. 

1.  Search and select the desired charge or service associated with a Payer Fee Schedule.

2.  From the Edit Charge slideout, select the Fee Schedules tab. 

3.  Update the desired amounts for all associated Fee Schedules, followed by SAVE (in blue). 

4.  Repeat the above steps for all desired charges or services.
