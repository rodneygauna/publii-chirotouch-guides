Patient Record - Billing

The Billing tab in a patient's profile consists of 5 subsections: General, Guarantor, Relationships, Packages & Subscriptions, and Recurring & Scheduled. 

General Tab

In General, you can set a default payment type (Self Pay or Insurance) or add a default payment method along with associating a Fee Schedule. 

Add Payment Type

Once a Payment Type is selected from here, appointments scheduled for the individual patient will default to the Payment Type selected. However, Payment Types can always be changed from the appointment if necessary. 

1.  Select either Self Pay or Insurance.

Note: An Insurance has to be first added to the patient's chart before you ca make this selection.

Add Payment Method

1.  Select Add Payment Method (in blue).

Note: Cards stored on file can be selected at a later date to easily process transactions without requiring a card swipe. Debit cards that require a PIN for processing cannot be stored.

2.  From the Add Payment Method dialog, enter the below details:

  • Card Description - helps to identify which card to use for service
  • Card Information - card number, expiration date, and CVV

3. Select SAVE PAYMENT METHOD (in blue). 

4.  View the added payment method.

Remove Payment Method

1.  From the payment method you wish to remove, select the minus (in blue).

2.  From the Remove Payment Method dialog, select YES (in blue).

Associate A Fee Schedule

Regardless of payment type, Fee Schedules can be associated and applied to patients.

  • For insurance based patients, Fee Schedules will automatically get applied once a payer associated with a Fee Schedule is added into the patient's record or selected when checking-in appointments.
  • For Self Pay patients, Fee Schedules can also associated and applied, providing a quick way to apply discounts across specific services and charges. 

Note: Fee Schedules must first be created in Settings. 

1. Select Associate Fee Schedule (in blue).

2.  From the Associate Fee Schedule slideout, select the desired Fee Schedule from the dropdown, followed by SAVE (in blue).

4. Once a Fee Schedule is associated with a patient, it will be available to select and apply to covered service and charges during Check In.

Guarantor Tab

By default, once a patient record is created, they are labeled as the primary guarantor, noted as the relation "self". In Guarantor, you can view, add, and delete a Patient Guarantor (including organizations).  

Add Patient Guarantor

1.  Select Add Patient Guarantor (in blue).

2.  From the Add Patient Guarantor slideout, choose Add Person. 

3.  Using the dropdown, choose the Relation to Patient.

4.  Select Create New Guarantor Record (in blue).

5.  From the Create Guarantor slideout, input the required details. Select SAVE GUARANTOR (in blue).

6.  From the Add Patient Guarantor slideout, update pertinent information. Select SAVE (in blue).

Add An Organization

1.  Select Add Patient Guarantor (in blue).

2.  From the Add Patient Guarantor slideout, choose Add Organization.

3.  Select Create New Organization (in blue).

4.  From the Add Organization slideout, input the required details. Select SAVE ORGANIZATION (in blue).

6.  From the Add Patient Guarantor slideout, update pertinent information. Select SAVE (in blue). 

Edit A Guarantor 

1.  Select the Guarantor you wish to edit.

2.  From the Update Patient Guarantor slideout, select EDIT (in blue).

3.  From the Update Organization slideout, update the pertinent information. Select SAVE ORGANIZATION (in blue).

Delete (Make Inactive)

1. Select the Guarantor you wish to delete (make inactive).

2.  From the Update Patient Guarantor slideout, select the Active toggle (in blue). 

3.  Select SAVE (in blue). 

Relationships Tab

You can use the Relationships tab to manage payments and billing between related patients.  Patients with relationships may share a credit card on file and have payments allocated between them.

1.  To set up a new relationship, click Add New Relationship (in blue).

2.  The Add Patient Relationship slideout will open.

3.  Set the Relationship To Patient and Search for an existing patient to link to this account.

4.  Once you select the patient to link, select the person that will be the Primary (usually the person that pays the bills or the primary on insurance) for this relationship. Click SAVE (in blue).

5.  The green banner will display and the linked patient will now show as a relationship.

The relationship will also show in the other patient's record.

Shared Card on File

If the relationship has a credit card on file, that card will be available for patients with a shared relationship. 

1.  If you Check Out a patient and go to make a payment, the shared card will be available to make this payment.

2.  Select the card to use for the payment from the Card On File dropdown.

Allocate Payment to Relationship

You may allocate payments between relationships.

1. If you select Allocate Payment, the Allocate Payments window will open and allow you to distribute payments to shared relationships.

Update or Inactivate Relationship

1. To update a relationship, select the far right arrow of the person you wish to update or make inactive.

2. Update the Relation to Patient, Primary relationship, add a NOTE, or change the slider to make the relationship Active (blue) or Inactive (gray). SAVE (in blue).

Packages & Subscriptions Tab

You can add and sell a service-based package to a patient, allowing them to utilize the package & subscription for their future visits. Below you will learn how to add, update and complete a package or subscription.

NOTE: Subscription Care Packages are designed as a standard subscription.

  • Visits and the price are set when the package was added
  • Will reset each month, on the day it was added

If you want to have a standard care package that does not have the visit count reset and would like to have the package paid off with monthly payments/installments, you can create a recurring payment through the Recurring & Scheduled tab within Billing on the patient's record.

Add A Package

1.  Select Add New Package (in blue).

2.  From the Add Patient Package/Subscription slideout, make your selections.

  • Visits / Services
    • e.g. Package - 5 visits for $100 or Subscription - 5 visits/month for $100
    • e.g. Package - 5 services for $100 or Subscription - 5 services/month for $100
  • Choose a Package Template.
  • To sell this package as a subscription, select the Sell as Subscription checkbox.
  • Select an Effective Date and Expiration Date.
  • Enter the Total Visits and Total Price of the package.
  • Review, or remove any preset charges or Add New Charges.

Note: Package Templates first have to be created in Billing Settings. Click here to learn more about Billing Settings - Package & Subscriptions. 

3.  Select Pay or Post.

  • Pay - allows the user to collect payment for the package or subscription which then can be posted to the patient's ledger
  • Post - allows the user to post the package or subscription to the patient's ledger without any payment collected

4.  View the added Package/Subscription.

Update A Package

1.  Select the package you wish to update.

2.  From the update Patient Package/Subscription slideout, edit the desired details and select SAVE (in blue).

Share A Package

You can only share a care package from the Primary Relationship.

1. Add a package and select the box for Share with Patient Relationships. Then from the dropdown, select the Patient(s) with whom to share the package.

2.  Shared packages can be seen in each relationship's billing tab.

Complete A Package

1.  Select the package you wish to mark as complete.

2.  From the update Patient Package/Subscription slideout, enter the Completion Date and select COMPLETE PACKAGE (in blue). 

Note: Once complete, charges may not be applied to this package, and any future scheduled payments for the package will be deleted.

3.  From the Complete Package dialog, select YES (in blue). 

4.  To view inactive packages, deselect the Hide inactive items checkbox.

Recurring Payments

In Recurring & Scheduled, you can add, view, and manage recurring or scheduled credit card payments. 

If you see this notice (below) when you click on Recurring & Scheduled, click here to learn how to set up your Merchant Account Services.

Schedule A Recurring Payment

1.  Select Schedule New Payment (in blue).

2.  From the Add Scheduled Payment slideout, select from the following options:

  • One-time - schedules a one-time payment for a desired date
  • Recurring - schedules a recurring monthly payment for a specific date range 
  • Payment Plan - schedules a recurring payment against a pre-set patient balance

Note: Cards stored on file can be used to easily process these transactions without requiring a card swipe. Debit cards that require a PIN for processing cannot be stored and can only be used by swiping with a card reader.

One-time Payment

From the Schedule One-time payment slideout, fill out all pertinent payment details.

Note: Payment Method requires a stored credit card on file. If there is not one on file, select Add Payment Method (in blue). 

Scheduled Recurring Payment

From the Schedule Recurring Payment slideout, fill out all pertinent payment details.

Scheduled Payment Plan

From the Scheduled Payment Plan slideout, fill out all pertinent payment details.

4.  Select SAVE (in blue).
