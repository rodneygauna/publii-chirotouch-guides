Charting 

Charting displays the selected provider's scheduled patient appointments for the day. From here, providers have the ability to add walk-in patients, chart encounters, review treatment plans, request next appointments, and much more. 

Charting Schedule

The Charting Schedule defaults to today's scheduled patient appointments for the selected provider. Users have the ability to adjust the date, the provider, start an encounter, and add walk-in patients. 

Charting Patient Card

The Charting Patient Card displays important appointment details.



The Appointment Symbols (shown below) give you a visual cue to the status of each patient on the schedule.

Expand the Patient Card

Use the blue arrow (on the right of patient card) to expand the Patient Card.  You can perform multiple functions from Charting.

  • Check In - displays until patient is Checked In

  • Check Out - displays once the patient has Checked In
  • Create Encounter - displays if no encounter has been created yet
  • View Encounter - displays once the encounter is created for this appointment
  • View Appointment Details - allows visible

Adjust The Charting Schedule Date & Providers

Schedule Date

Providers can see future and past scheduled appointments from Charting. 

1. From Charting, select the right or left arrow at the top of the schedule (in blue). 

Providers

Select which provider's scheduled patient appointments you wish to see by using the select providers dropdown.

1. Select the desired provider from the Select providers dropdown.

Add Walk-In Patient

Easily add Walk-In Patients in Charting. Walk-In patients do not account for provider availability or patients allowed per booking slot. 

1. From Charting, select Add Walk-In Patient at the bottom of the schedule (in blue).

2. From the Add Appointment slideout, fill in the details below:

  • Search for a Patient - search for a patient using names, MRN, DOB, phone number
  • Add New Patient - allows you to add a new patient
  • Provider - select the desired provider
    Note: This will default to single-provider practices
  • Appointment Type - select the type of appointment
  • Case - select the patient-related case, if applicable
  • One-Time Appointment - will be selected by default 
  • Select by - will default to the override option, allowing you to bypass provider availability and patient bookings per time slot
  • Select time - will default to the current hour

3. Select SCHEDULE (in blue).

4.  The newly added patient will display in the schedule at the appropriate time.
